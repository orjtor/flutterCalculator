# Flutter Calculator

En enkel kalkylator byggd i Flutter.

## Funktioner

- Grundläggande räkneoperationer (+, -, ×, ÷)
- Display med historik och aktuellt värde
- Decimaler
- Clear-funktion


## Koncept som används

- StatefulWidget & StatelessWidget
- State management med `setState()`
- Widget composition
- Callbacks för event-hantering
- Column & Row layout
- Material Design

## Kör projektet

```bash
flutter run
